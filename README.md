## ✨ Kluczowe Funkcje

* **⚡ Interaktywny Generator Treningu:** Unikalne narzędzie pozwalające wybrać podział treningowy (np. Split, Góra/Dół) oraz partie mięśniowe, aby wygenerować konkretny plan.
* **📋 Strefa Treningowa:** Szczegółowe instrukcje ćwiczeń z rozpisanymi powtórzeniami, tempem, przerwami oraz licznikiem ukończonych serii.
* **📰 System Aktualności:** Panel informujący o życiu siłowni i ważnych wydarzeniach.
* **📜 Cyfrowy Regulamin:** Przejrzysty spis zasad panujących w obiekcie, dostępny dla każdego użytkownika.
* **📩 Formularz Kontaktowy:** Prosty sposób na zadanie pytania kadrze siłowni.
* **📱 Mobile First:** Strona w pełni zoptymalizowana pod urządzenia mobilne.

## 🛠️ Technologie
Projekt został zbudowany przy użyciu nowoczesnych narzędzi webowych:
* **Frontend:** React / JavaScript
* **Style:** Tailwind
* **Hosting:** Netlify

## 🏗️ Struktura Strony
1.  **Start** – Sekcja Hero z hasłem przewodnim.
2.  **O siłowni** – Historia i misja obiektu.
3.  **Regulamin** – Zasady wstępu i bezpieczeństwa.
4.  **Generator treningu** – Narzędzie do generowania treningu.
5.  **Kontakt** – Sekcja z formularzem kontatkowym.
